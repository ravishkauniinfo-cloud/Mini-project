Number System Converter

Convert between binary, decimal, octal, and hexadecimal

Number Systems Explained:

Binary (Base 2): Uses digits 0-1. Used in computers. Example: 1010₂ = 10₁₀
Octal (Base 8): Uses digits 0-7. Used in computing. Example: 12₈ = 10₁₀
Decimal (Base 10): Standard numbering system. Uses digits 0-9. Example: 10₁₀
Hexadecimal (Base 16): Uses digits 0-9 and letters A-F. Example: A₁₆ = 10₁₀

How to Use:

Step 1: Select the input NUMBER SYSTEM (e.g., Decimal)
Step 2: Enter the number (e.g., 255)
Step 3: Select the output system (e.g., Binary)
Step 4: Click Convert
Result: 11111111

Examples:
• Decimal 100 → Binary: 1100100
• Decimal 255 → Hexadecimal: FF
• Binary 1010 → Decimal: 10
• Hexadecimal ABCD → Decimal: 43981

🔧 Pro Tip: Click on the binary representation to toggle individual bits (0→1) for educational purposes

Calculus Tools

Derivatives, integrals, limits, and series operations

Calculus Operations:

Derivatives: Find the rate of change of a function. Example: d/dx(x²) = 2x
Limits: Determine the value a function approaches. Example: lim(x→∞) 1/x = 0
Series: Sum of sequences of terms. Example: Taylor series, Fourier series
Integration Concepts: Anti-derivatives and area under curves

How to Use:

Derivative Example:
Step 1: Select "Derivative"
Step 2: Enter function: x^3 + 2x
Step 3: Specify variable: x
Step 4: Get result: 3x² + 2

Limit Example:
Step 1: Select "Limit"
Step 2: Enter function: sin(x)/x
Step 3: Set approach value: 0
Step 4: Get result: 1

Series Example:
Taylor Series of sin(x) around 0:
x − x³/3! + x⁵/5! − x⁷/7! + ...

📚 Key Concepts:

f'(x) — First derivative (slope/rate of change)
f''(x) — Second derivative (concavity)
∫f(x)dx — Indefinite integral (antiderivative)
∫ₐᵇf(x)dx — Definite integral (area under curve from a to b)
